# Moho-Animation
Moho Animation website developement

1. Download or clone the repo
2. Open 'config.yml' and change the 'theme_id' to your development theme. Also you may need to change the 'password' and the 'store' if you are using different store. 
3. It has some npm scripts which allows to run the themekit commands. You will find them on the 'package.json' file. You can run the commands from the terminal or you can just run from the list under 'NPM SCRIPTS' on the left sidebar of vs code.
